開機自動啟動(使用systemd):/home/raslan/Desktop/lidar_lan_ras3.py
sudo nano /etc/systemd/system/lidar_lan_ras3.service

貼上內容如下："

[Unit]
Description=Start lidar_lan_ras3
After=network.target

[Service]
ExecStart=/usr/bin/python3 /home/raslan/Desktop/lidar_lan_ras3.py
WorkingDirectory=/home/raslan/Desktop
StandardOutput=inherit
StandardError=inherit
Restart=always
User=raslan
Environment=DISPLAY=:0
Environment=XAUTHORITY=/home/raslan/.Xauthority

[Install]
WantedBy=multi-user.target

"
啟用 service 服務
sudo systemctl daemon-reexec
sudo systemctl daemon-reload
啟用創建的.service
sudo systemctl enable lidar_lan_ras3.service
立即啟動（選擇性）：
sudo systemctl start lidar_lan_ras3.service
OK
----------------


sudo systemctl status lidar_lan_ras3.service
sudo systemctl stop lidar_lan_ras3.service
 其他也類似，修改 lidar_lan_ras3 部分即可

取消開機自動啟動：
sudo systemctl disable lidar_lan_ras3.service
重新載入 systemd 服務列表：
sudo systemctl daemon-reload
確認服務不會再自動啟動：
systemctl is-enabled lidar_lan_ras3.service
查看目前 子動啟動 service 列表
systemctl list-units --type=service

資料夾修改成英文
1.執行以下指令，把所有 XDG 資料夾設成英文名稱：

xdg-user-dirs-update --set DESKTOP "$HOME/Desktop"
xdg-user-dirs-update --set DOWNLOAD "$HOME/Downloads"
xdg-user-dirs-update --set TEMPLATES "$HOME/Templates"
xdg-user-dirs-update --set PUBLICSHARE "$HOME/Public"
xdg-user-dirs-update --set DOCUMENTS "$HOME/Documents"
xdg-user-dirs-update --set MUSIC "$HOME/Music"
xdg-user-dirs-update --set PICTURES "$HOME/Pictures"
xdg-user-dirs-update --set VIDEOS "$HOME/Videos"

xdg-user-dirs-update


2.將舊的「中文資料夾」移到英文資料夾：

mv ~/桌面/* ~/Desktop/ 2>/dev/null
mv ~/文件/* ~/Documents/ 2>/dev/null
mv ~/下載/* ~/Downloads/ 2>/dev/null
mv ~/音樂/* ~/Music/ 2>/dev/null
mv ~/圖片/* ~/Pictures/ 2>/dev/null
mv ~/影片/* ~/Videos/ 2>/dev/null
mv ~/公共/* ~/Public/ 2>/dev/null
mv ~/範本/* ~/Templates/ 2>/dev/null


3.如果確認資料都搬好了，也可以把原本的中文資料夾刪掉：

rmdir ~/桌面 ~/文件 ~/下載 ~/音樂 ~/圖片 ~/影片 ~/公共 ~/範本 2>/dev/null


4.
cat ~/.config/user-dirs.dirs

你應該會看到像這樣的結果（以英文為主）：
XDG_DESKTOP_DIR="$HOME/Desktop"
XDG_DOWNLOAD_DIR="$HOME/Downloads"
XDG_DOCUMENTS_DIR="$HOME/Documents"
...

}按下 Ctrl + O 儲存，然後 Ctrl + X 離開。
